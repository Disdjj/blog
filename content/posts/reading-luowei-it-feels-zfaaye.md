# 读<筚路维艰>有感

[书籍链接](https://book.douban.com/subject/26171466/)

我向来是不喜欢假大空的内容, 我从其中没法获得真正的养料.

偶然间看到这本书, 就耐着性子读了读. 确实收获颇丰, 这可能也和刚刚读完矛盾论和实践论有关系.

坦白说作者还是夹带着私货的, 但是总体上说还是客观而且史料清晰的.

# 主席

我对毛泽东的崇拜由来已久, 实际上是从我爷爷开始, 那是出于一种朴素的热爱.

我喜欢诗歌, 尤其是气质豪迈的诗歌, 辛弃疾, 主席自然是个中好手.

但是稼轩终究是一生都不得施展抱负, 词当然是豪迈的, 但是读了背景史料又不免郁结.

主席则不一样, 他的诗词有着巨大的革命热情和乐观精神, 一生革命, 诗词中杀气甚重.

我热爱而且崇拜他, 这种热爱和崇拜会随着了解的深入和去神像化后更加激烈.

该怎么评价主席呢?

1. 爱国者
2. 无产主义者
3. 革命者
4. 自我革命者

我们必须得承认一点, 主席确实将自己的一生献给了中国, 是非功过后人评价了许多. 但是我必须要说的是, 我认为他的一切决定都是为公的. 但是在一些决策上是缺少事实支撑导致做出了错误的决定.

阅读了这本书之后, 作者夹带私货想要将毛主席塑造为一个独裁者, 这部分的观感不佳, 望改正. 不改正就他妈给你两拳.

# 革命/自我革命

革命是什么? 我认为革命是对社会关系/生产关系/政治关系的一种颠覆性变革.

如果阅读了矛盾论就会知道, 斗争是矛盾的一种表现形式.

革命第一步则是要找到矛盾. 革命者只会捍卫自身阶级的利益, 所以革命阶级的群体越庞大, 革命的影响力就越大.

为什么主席强调人民的力量, 因为这是最庞大的革命阶级.

文化大革命为什么会演变的如此浩荡? 仅仅是因为毛主席和所谓的高层的推波助澜?

需要知道一件事情, 革命的烈度和社会矛盾的尖锐程度是直接相关的.

文革前的社会矛盾并没有消失, 只是被掩盖掉了.

自我革命是一个更加深刻的问题. 一个组织真的能够进行自我革命吗? 难!

庞大的利益阶级, 层层的权力负责体系, 体制外的压力以及利益往来, 顶层领导者闭塞的信息渠道.

这些都为成为自我革命的阻碍.

我党还是有魄力的, 但是几十年下来庞大的利益层级关系实在太复杂了, 我对这件事情是否立刻发生报有怀疑, 但我毫不怀疑它一定会发生, 或者正在发生.

如果一个组织不能自我革命, 那么在扩张结束之后只会迎来灭亡. 

# 权力

权力来自于控制, 这种控制可能是暴力的也有可能是非暴力的. 暴力的如警察, 军队. 非暴力的如住房, 医生.

但是归根到底, 权力来自于暴力保证.

权力是可以进行分化的, 高级别的权力可以产生子级别的权力.

权力只会对权力的来源负责, 这句话越体会越深刻. 这种负责有好有坏:

1. 保证执行力
2. 权力的滥用
3. 导致信息阻塞

最坏情况下, 权力不会对自己的实际责任对象做出任何有利的事情, 进而转变为剥削阶级.

这也是为什么文革要发动群众. 被剥削阶级总是会倾向于更大的利益获取, 而利益总是要侵害剥削阶级的既得利益, 间接的实现了监督的作用.

# 剥削阶级

在此之前, 马列说是资产阶级, 毛说是变质的官僚.

但是可以概括为一类: 在社会体系下通过侵占剩余价值的剥削阶级.

所有的被剥削阶级都是无产者, 你的劳动无法换取正常的价值, 你所得的价值也是被人抹去了一层油脂.

剥削阶级是贪婪腐败的, 而且会成组织, 成体系, 成世代.

这是剥削阶级对社会最大的危害.

很多人讲, 贪官不要紧, 怕的是又贪又没用. 这句话是完全错误的, 长远看一个有能力的贪官和一个没能力的贪官只有程度山的区别, 却没有本质上的区别.

本质上这些贪官都是在售卖公权力, 导致民众信任下降, 营商环境的败坏. 打贪腐本质上就是在自救.

剥削阶级为了掩饰剥削的事实, 会用一些小的无关紧要的事情进行舆论扩大, 来营造自己的形象. 假的一比.

# 搜集事实, 敢用事实说话

为什么要搜集事实?

如果从实践论的观点看, 没有事实依据做出判断的话, 只能走向错误的道路.

怎么搜集事实?

1. 真实有效
2. 底层反馈

在现在的网络环境下, 底层反馈这件事情变得越来越广泛, 但是真实性和有效性都需要做好鉴别, 尤其是注意剥削阶级以此为理由进行封堵.

用事实说话.

在拥有事实之后, 我们才能获得全面的感性认识, 有了认识才能形成理论.

事实和实践是不断变化的, 所以理论也要不断变化, 才能适应形势.

敢用事实说话.

指的是在形势不利的情况下, 也要感觉说出真相, 避免形成错误的感性认识, 得到错误的结论, 进而导致了下一步的错误实践.

错误是会累积的.

# 有底线的实用主义革命

1. 任何能够解决当前问题的方案都应该考虑
2. 方案的得到要考眼睛和手
3. 绝对不要想着一次性将问题全部解决
4. 要保留纯洁的底线

这是我从邓小平的改革中学习到的.

我认为这是最有价值的地方: 在保证纯洁性的基础上, 灵活的运用不同手段, 解决当前问题.

这对应对工作中的问题是有巨大帮助的.
