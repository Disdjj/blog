## Hugo 简介

Hugo 是一个用 Go 语言编写的静态网站生成器，具有以下特点：

- **速度快** - 构建速度极快
- **易用** - 简单的 Markdown 语法
- **灵活** - 丰富的主题和自定义选项
- **免费** - 完全开源

## 代码示例

Hugo 支持语法高亮，比如这段 Python 代码：

```python
def hello_world():
    print("Hello, World!")
    print("欢迎来到我的博客")

if __name__ == "__main__":
    hello_world()
```
